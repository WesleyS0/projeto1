Projeto 01 (21/08)

Este projeto é um desafio para a criação de projeto com o Spring Boot, onde devem haver 3 rotas diferentes e pelo menos duas variáveis.

E pensando nisso, decidimos fazer um sistema de notas, onde o usuário vai entrar com um nome, e duas notas. Dependendo da média, o usuário será levado para "aprovado" ou "reprovado"."
